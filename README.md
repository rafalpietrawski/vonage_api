# vonage_api test

## Requirements

* [Python 3.11.x](https://www.python.org)

## Setup

Install and activate Virtual Environment

### Create venv

```shell
make venv-create 
```

### Activate venv

```shell
make venv-activate
```

### Install `requirements`

```shell
make setup
```

### Create .env

Edit it according to given example in `.env_example`


# Run

```shell
make run
```
